# Preflight Research
Research project exploring AI Agent Security, MCP Security, Prompt Injection, Attack Graph Analysis, and Permission Risk Prediction for autonomous AI systems.

## Research Goal

Current AI agents are increasingly connected to tools, databases, browsers, file systems, APIs, and external services. Existing security approaches primarily detect attacks after deployment.

PreFlight investigates whether risky agent configurations and attack paths can be identified before deployment using permission analysis, attack graph modeling, and machine learning.

## Research Areas

* AI Agent Security
* MCP Security
* Prompt Injection
* Tool Permission Analysis
* Attack Graphs
* AI Safety

# Finalist Comparison

## Objective

Identify the strongest startup-grade opportunity among the three finalists:

1. PreFlight
2. ThreatBench
3. TaskAnchor

Evaluation Criteria:

* Market Opportunity
* Startup Potential
* Research Novelty
* Recruiter Value
* Technical Feasibility
* Smart India Hackathon Potential
* MVP Feasibility

---

# Finalist 1: PreFlight

## Summary

PreFlight is an AI Agent Permission Simulation and Risk Analysis Platform.

Before an AI agent is deployed, PreFlight evaluates:

* Agent permissions
* Tool access
* API exposure
* Potential attack paths
* Data leakage risks

## Customer

* Enterprises deploying AI agents
* SaaS companies
* AI platform providers

## Strengths

* Growing market
* Strong security need
* High startup potential
* Relevant to agentic AI trend

## Weaknesses

* Requires realistic agent execution modeling
* Security simulation complexity

# Finalist 2: ThreatBench

## Summary

ThreatBench is a benchmarking and evaluation platform for AI agent security.

It measures:

* Prompt injection resistance
* Tool abuse resistance
* Data exfiltration risk
* Agent robustness

## Customer

* AI companies
* Security researchers
* Enterprises adopting AI

## Strengths

* Strong research value
* Clear benchmarking problem
* Publication opportunities

## Weaknesses

# Finalist 3: TaskAnchor

## Summary

TaskAnchor provides runtime governance for autonomous AI agents.

The platform continuously monitors:

* Agent actions
* Goal drift
* Tool misuse
* Policy violations
* Risk escalation

## Customer

* Enterprises deploying autonomous agents
* Security teams
* AI platform providers

## Strengths

* Directly addresses future AI governance needs
* Strong enterprise value
* Long-term strategic relevance

## Weaknesses

* Most technically challenging
* Requires sophisticated monitoring architecture

# Key Observation

PreFlight and TaskAnchor address real enterprise security pain points and have clearer monetization opportunities.

ThreatBench is valuable but appears better suited as a research platform or supporting product rather than a standalone startup.

TaskAnchor represents the largest market opportunity but also carries the highest execution risk.
